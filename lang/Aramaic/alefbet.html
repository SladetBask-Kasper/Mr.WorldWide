What does "Alef bet" mean?

So "Alef bet" is just the name of the first two letters of the alphabet. Just like our word in english "Alphabet" is just the name of the first to letters of the Greek alphabet (which the roman one is based upon): Alpha Beta.

Let's look at some text in Syriac


ÜÜÜ£ÜªÜ³ÜÜ¶Ü 

This word starts with Aleph, which in this case is pronounced kinda like the double e in "meet", the "ee" sounds kinda like an "i" does in other languages and that is important to know.

The next word is the letter Yod. This is what makes a Aleph into an "i" sound. So Aleph, Yod, in that order becomes like an "i" sound.

The next letter is Semkath, which is just an "S" sound. So we so far have the word as "is". And the one after that is RÄ“sh which is just an "R" sound. so, so far we have "Isr".

The next letter is yet another Yod. In this case that is kinda an "a" or "i", sound, depending on how you want to pronounce it. Both pronounciations are kinda wrong and approximations in this case, and a little bit wrong. "Isra".

The last letter is really easy to remeber. It's LÄmadh, and it looks just like our upper case "L" but reversed, which kinda fits when it's written from right to left. If I was to write English from right to left I'd probably write it in reverse so it's quite natural to read it like that.

Solution: The word is "Israel".

The first letter is once again Aleph. The second letter is Resh. The next letter is Mem. We will when pronouncing it with an "a" sound between the Resh and Mem. So so far we have "Aram".

Yod is the next letter, it is a tricky letter. It can be pronounced as alot. I/E/Y/EE/and so on... In this case I'll transliterate it as an y for simplicity sake. You'll probably already now see what word it is. So far we have "Aramey".

The final letter is Aleph. So "Arameya" is what I will transliterate this word as. Which means Aramean. It can also mean Syrian as the Syriac people still call themselves Aramaian.

Solution: The word is "Aramean".
